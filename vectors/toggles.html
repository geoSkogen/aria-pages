Toggle Types

Simple Image Toggle: CSS-only, no custom HTML

Use when there is no access to the HTML, e.g., formulaic CMS rendering.

Override the checkbox appearance with an image URL in its content property. Toggle the image URL using the :checked pseudo-selector

Plain Toggle: CSS and custom HTML

Requires enough control over the HTML to enclose the checkbox in its own label element.

Enable notifications:

Wrap the checkbox in its own label & minimize the size of the checkbox, so the label becomes the click target. Append the label with a div and transform/transition the absolute position of its :before pseudo-element based the checkox's :checked pseudo selector. Ensure the checkbox's aria-labelledy refers to a relevant text element.

"Ghost" Toggle: CSS, some custom HTML, & JavaScript

Use when custom HTML can be rendered around the field but the input and label elements are governed by formulaic CMS rendering.

Enable notifications:

Assign a tabindex and event listener to a plain div and use the absolute position of its :before pseudo-element to represent the toggle state. Because div isn't a form element, utilize JavaScript to maintain its state and simultaneously transfer its state to to a hidden checkbox
